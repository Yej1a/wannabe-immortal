# Wannabe Immortal

原创作者：Yej1a

`Wannabe Immortal` 是一个修仙题材网页游戏原型项目。

## 在线试玩

网页地址：

- [https://yej1a.github.io/wannabe-immortal/](https://yej1a.github.io/wannabe-immortal/)

打开网页后，点击 `开始试炼` 即可进入游戏。

## 怎么玩

基础操作：

- `WASD` / 方向键：移动
- 术法会自动施放
- `1` / `2` / `3`：释放已解锁的主动术法
- `Q` / `E`：释放黑白槽满后的战斗能力
- `F`：切换全屏
- 点击 `暂停`：暂停战斗

基础流程：

- 开局先从命格和术法中建立自己的构筑
- 战斗中通过击杀敌人获取修为升级，选择术法强化
- 同一术法强化到一定次数后，会出现不同分路可选
- 击败小 Boss 后会进入升级、点化和整备阶段
- 继续推进关卡，完成整轮试炼

界面说明：

- 左侧显示生命、修为、状态和命格
- 中间是战场与技能栏
- 右侧显示白槽、黑槽和当前提示
- 悬停或点击状态、术法、命格，可以在详情面板查看构筑信息

## 本地运行

如果你想在本地打开项目，可以直接启动一个静态服务器，例如：

```powershell
py -3 -m http.server 4173
```

然后访问：

- [http://127.0.0.1:4173/](http://127.0.0.1:4173/)
